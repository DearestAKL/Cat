# 猫国建设者

一款基于网页的增量/放置类游戏，复刻自经典游戏"Kittens Game"。

## 游戏说明

从零开始建设你的猫咪王国！通过采集资源、建造建筑、研究科技来发展你的猫咪帝国。

### 核心玩法

1. **采集资源**：点击采集猫薄荷，转化为木材等基础资源
2. **建造建筑**：建造猫薄荷田、帐篷、图书馆等设施
3. **分配职业**：安排猫咪从事伐木、科研、狩猎等工作
4. **研究科技**：解锁各种科技提升生产效率
5. **贸易系统**：与其他种族交易获取稀有资源
6. **宗教系统**：积累信仰获得永久加成
7. **轮回重置**：获得领导力和业，加速下一轮发展

### 资源系统

- **基础资源**：猫薄荷、木材、矿物、煤炭、铁、钛
- **高级资源**：毛皮、羊皮纸、文化、信仰
- **特殊资源**：科技点、领导力、业

### 建筑系统

- **猫薄荷田**：自动产出猫薄荷
- **帐篷**：增加猫咪人口上限
- **图书馆**：允许分配学者产出科技点
- **剧场**：提升猫咪幸福度
- **工作台**：解锁制作功能
- **矿井**：产出矿物
- **庙塔**：产出信仰和文化

### 职业系统

- **伐木工**：产出木材
- **学者**：产出科技点
- **猎人**：产出毛皮
- **矿工**：产出矿物
- **牧师**：产出信仰

### 季节系统

游戏包含四季循环，不同季节对猫薄荷产量有影响：
- 春季：正常产量
- 夏季：+50% 产量
- 秋季：正常产量
- 冬季：-25% 产量

### 轮回系统

当猫咪数量达到70只以上时，可以重置轮回：
- 获得领导力（每超过70人口+1）
- 获得业（超过35人口时获得）
- 领导力提供永久生产加成
- 业提供幸福度加成

## 技术实现

- 纯HTML/CSS/JavaScript实现
- 遵循DESIGN.md设计规范（Ollama风格）
- 使用LocalStorage保存游戏进度
- 响应式设计，支持移动端

## 设计规范

本项目严格遵循DESIGN.md中定义的设计系统：

- **颜色**：严格灰度（纯黑#000000到纯白#ffffff）
- **字体**：SF Pro Rounded用于标题，ui-sans-serif用于正文
- **圆角**：二元系统 - 12px（容器）或9999px（交互元素）
- **阴影**：无
- **字重**：仅400和500

## 如何部署

1. 登录GitHub账号
2. 进入仓库设置
3. 找到GitHub Pages设置
4. 选择部署来源为main分支
5. 保存设置后，等待几分钟即可访问

## 操作说明

- 点击顶部按钮保存/读取/重置游戏
- 使用标签页切换不同功能模块
- 游戏每30秒自动保存一次
